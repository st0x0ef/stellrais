# Getting Started

Welcome to the wiki of the Stellaris Project !

## Install the mod

The Stellaris Project is currently available for Minecraft `1.20.6` and `1.21`/`1.21.1`

| JEI              |                                  [Curseforge](https://www.curseforge.com/minecraft/mc-mods/jei)                                  |  false  |
